Chart LLM Resources

Link to Chart Dataset

You can access the chart dataset at the following link: Chart Dataset .

Please note that the datasets are not images, but Vega-Lite specifications. You may visualize and save them as PNG, using the Vega-Lite editor .

NL Dataset

We provide sample results of L1/L2 captions, utterances, and questions for 48 charts selected via stratified sampling. You can access the results from the following links:

  • L1: L1_Description.JSON
  • L2: Task1_L2.CSV (column: L2)
  • Utterance: Task2.CSV (columns: commands, queries, questions)
  • Question: Task3.CSV (columns: lookup-nonvisual, lookup-visual, compositional-nonvisual, compositional-visual, open-ended)
